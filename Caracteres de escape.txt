Caracter de Escape   Usa quando precisa de...                         Exemplo de uso


\n                   Quebra de linha (nova linha)                     "Linha 1\nLinha 2"
-------------------------------------------------------------------------------------------------------------
\t                   Tabulação (espaço horizontal)                    "Coluna1\tColuna2"
-------------------------------------------------------------------------------------------------------------
\\                   Barra invertida (para inserir uma \ literal)     "C:\\Users\\Gabriel"
-------------------------------------------------------------------------------------------------------------
\'                   Apóstrofo simples dentro de string com aspas simples   'It\'s legal'
-------------------------------------------------------------------------------------------------------------
\"                   Aspas duplas dentro de string com aspas duplas   "Ele disse: \"Oi\""
-------------------------------------------------------------------------------------------------------------
\r                   Retorno de carro (volta ao início da linha)       "12345\rabc"  # sobrescreve começo
-------------------------------------------------------------------------------------------------------------
\b                   Backspace (apaga caractere anterior)              "Hello\bWorld"  # Apaga o 'o'
-------------------------------------------------------------------------------------------------------------
\f                   Alimentação de formulário (form feed)              "Texto\fNovaPágina"
-------------------------------------------------------------------------------------------------------------
\a                   Campainha / alerta sonoro (pode não funcionar em todos ambientes) "Beep\a"
-------------------------------------------------------------------------------------------------------------
\ooo                 Representação octal de caracteres                   "\101"  # letra 'A'
-------------------------------------------------------------------------------------------------------------
\xhh                 Representação hexadecimal de caracteres             "\x41"  # letra 'A'
-------------------------------------------------------------------------------------------------------------
